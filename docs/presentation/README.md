# Presentation Packet

Last updated: June 13, 2026.

This folder holds the presentation planning materials for the CrisisCoord hackathon submission.

## Files

- [gamma-presentation-plan.md](./gamma-presentation-plan.md): Gamma research notes, prompt, slide outline, architecture caveat, and team meeting questions.
- [crisiscoord-presentation-planning-workbook.xlsx](./crisiscoord-presentation-planning-workbook.xlsx): spreadsheet workbook for the team meeting, deck planning, problem/solution mapping, demo flow, partner proof, and provisional architecture decisions.

## Rule

The deck may explain the architecture concept, but it must clearly mark detailed architecture as provisional until the team finalizes implementation choices.

## Gamma Workflow

1. Open Gamma.
2. Start from pasted text or imported content.
3. Use the prompt in [gamma-presentation-plan.md](./gamma-presentation-plan.md).
4. Generate the first deck.
5. Review all compliance and architecture claims manually.
6. Add app screenshots only after the UI exists.
7. Export final slides to PDF/PPT and keep the Gamma link for submission.
